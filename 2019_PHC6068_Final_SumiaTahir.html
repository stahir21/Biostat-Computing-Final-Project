Data description.

Background

Physical fitness researchers were interested in whether following a new program including both a regimen of exercise and special diet would lead to young men with an interest in weight-lifting to be able to bench press greater amounts of weight and to do it more quickly than if they were to follow only the exercise part of the program alone. Thus, they had a particular interest in the effects of the diet portion of the program.

This figure is from https://healthhabits.wordpress.com/marge-simpson-bench-press-2/

To investigate, the researchers recruited 100 young men in high school, college, and beyond with either existing interest and experience with weight-lifting or interest in becoming involved in weight-lifting. It is well-known that the amount of weight a man can bench press may be associated with their body weight, previous weight-lifting experience, and age. Thus, the researchers recorded these baseline characteristics for each man. The men were randomized at the beginning of the study to 2 groups, 50 men per group: (1) Follow the exercise part of the program only; (2) Follow both the exercise and diet parts of the program.

The amount of weight each man was capable of bench pressing at entry into the study was recorded for all men (day 0). Subsequently, the men were allowed to come to the gym at which the study was conducted according to their own schedules, as would be the case in practice. Periodically, members of the research staff would record the amount (lbs) each man was able to bench press (the response). Because each mans schedule was different due to their class or work obligations, the times at which this was recorded for each man varied across men. Most men were followed for about 9-10 months.

Dataset

The prepared data file is available at https://Caleb-Huo.github.io/teaching/data/pressData/benchPress.csv. Your analysis should be based on this data.

Dataset characteristics

  • ID: participant id
  • time: in terms of days (since entry into the study) for bench press record
  • press: the amount of bench press (in lbs)
  • weight: the body weight of the participant at the beginning of study
  • age: the age of the participant at the beginning of the study
  • prev: indicator for whether the participant has the previous weight lifting experience (0: no; 1: yes)
  • diet: indicator for whether the participant is not in the diet program (0) or in the diet program (1)

Question 4. (10 pts)

Use permutation test to obtain the p-value for the diet program of the same statistical model. Hint: for the permutation test, the null hypothesis is that the distributions of the outcome variable (i.e., improvement in bench press) are the same for the both diet programs. With that being said, under the null, by randomly shuffling the diet program status among all the participants, the null distribution of the coefficient estimate of diet remains the same. Therefore, for each permutation iteration, we can use the coefficient estimate of diet as the test statistic. By permutation \(B=1,000\) times, we can get a pool of null test statistics, which will be used as the null distribution of the coefficient estimate of diet. The p-value of diet (we can consider one-sided test) from the permutation test can be obtained by comparing how extreme the observed test statistics is among its null distribution.

T_obs <- coef(lm_q2)["diet"]



B <- 1000

TB <- numeric(B)

for(b in 1:B){
  set.seed(b)
  bdataFrame <- df_q2
  bdataFrame$diet <- sample(bdataFrame$diet)
  lm_b <- lm(improvement ~ weight + age + pre + diet + press_ini, data = bdataFrame)
  TB[b] <- coef(lm_b)["diet"]
}


p_perm <- sum(TB >= T_obs)/B
print(paste("Permutation p-value:", p_perm))
## [1] "Permutation p-value: 0"

The p-value given by the permutation test is zero which means that the observed statistic is greater than all of the null test statistics; therefore, we reject the null hypothesis that the diet groups have the same distribution. The improvement in bench press weight is not the same in the diet vs no-diet group.

Based on the problem setting in Question 5, perform 10 fold cross validation to select the best tuning parameter \(\lambda\) using all samples. Pre-specify the range of \(\lambda\) to be \((0,0.1,0.2, \ldots, 15)\). Use mean squared error (MSE) as the criteria make the decision. Visualize the curve of MSE versus \(\lambda\). Mark the \(\lambda\) with the minimum MSE using a different color in the curve. What is this optimum \(\lambda\)? Under this \(\lambda\), how many covariates will have non-zero coefficient in a model with all subjects? Reminder: dont forget to set normalize = FALSE

K <- 10
folds <- cut(seq(1,nrow(df_q5)),breaks=K,labels=FALSE)
lambdas <- seq(0,15,0.1)

MSEs <- matrix(NA,nrow=length(lambdas),ncol=K)
rownames(MSEs) <- lambdas
colnames(MSEs) <- 1:K

for(k in 1:K){
  atrainx <- as.matrix(df_q5[!folds==k, 2:6])
  atestx <- as.matrix(df_q5[folds==k,2:6])
 
  atrainy <- df_q5[!folds==k,7]
  atesty <- df_q5[folds==k,7]

  klassoFit <- lars(atrainx, atrainy, normalize=FALSE) 

  predictValue <- predict.lars(klassoFit, atestx, s=lambdas, type="fit",  mode="lambda")
  MSE <- apply(predictValue$fit,2, function(x) sqrt(sum((x - atesty)^2)))
  
  MSEs[,k] <- MSE
}

MSEdataFrame <- data.frame(lambda=lambdas, MSE = rowMeans(MSEs))
ggplot(MSEdataFrame) + aes(x=lambda, y = MSE) + geom_path() + ggtitle("MSE vs lambda using 10-fold CV") + 
  geom_point(data=MSEdataFrame[which.min(MSEdataFrame$MSE),], aes(x=lambda, y=MSE, col="red"))

opt_lambda <- with(MSEdataFrame, lambda[which.min(MSE)])
print(paste("Optimum lambda:", opt_lambda))
